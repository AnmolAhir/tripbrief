## Deploy to Azure Static Web Apps (Student Account)

### Prerequisites
- GitHub account
- Azure account (https://azure.microsoft.com/free/students)

### Step 1 вЂ” Push to GitHub
```bash
git init
git add .
git commit -m "first commit"
```
Go to github.com в†’ New Repository в†’ name it `tripbrief` в†’ Create

```bash
git remote add origin https://github.com/YOUR_USERNAME/tripbrief.git
git branch -M main
git push -u origin main
```

### Step 2 вЂ” Create Static Web App on Azure
1. Go to https://portal.azure.com
2. Search for **Static Web Apps** в†’ Click **Create**
3. Fill in:
   - **Subscription**: Azure for Students
   - **Resource Group**: Create new в†’ name it `tripbrief-rg`
   - **Name**: `tripbrief`
   - **Region**: Central US (or closest)
   - **Source**: GitHub
4. Click **Sign in with GitHub** в†’ Authorize
5. Select:
   - **Organization**: your GitHub username
   - **Repository**: tripbrief
   - **Branch**: main
6. **Build Details**:
   - Build Preset: **Vite**
   - App location: `/`
   - Output location: `dist`
7. Click **Review + Create** в†’ **Create**

### Step 3 вЂ” Wait ~2 minutes
Azure auto-triggers a GitHub Action that builds and deploys your app.
You'll get a free URL like: `https://xyz.azurestaticapps.net`

### Step 4 вЂ” Done! рџЋ‰
Every time you `git push`, Azure auto-redeploys. No manual steps needed.
